# OP_FlashLoan ⚡

**The first flash loan protocol on Bitcoin Layer 1, powered by OP_NET.**

![OP_FlashLoan](https://bitcoin-flashloan.vercel.app)

## What is OP_FlashLoan?

OP_FlashLoan brings the most powerful DeFi primitive — flash loans — to Bitcoin L1 for the first time. Borrow any amount from the liquidity pool, execute your strategy, and repay in a single atomic transaction. If repayment fails, the entire transaction reverts.

## Features

- ⚡ **Atomic Flash Loans** — Borrow, execute, repay in 1 transaction
- 🏦 **Liquidity Pools** — Deposit BTC and earn ~8.4% APR from loan fees
- 🛡️ **Zero Default Risk** — Atomic revert protection for all LPs
- 💰 **0.09% Fee** — Competitive fee distributed to liquidity providers
- 🔗 **Bitcoin L1** — No bridges, no wrapped tokens, native Bitcoin
